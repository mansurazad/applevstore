# Tauri Icons

Place generated icons here before building. Easiest way:

```bash
npx @tauri-apps/cli icon ./public/lovable-uploads/3926e988-d85b-4bf1-8f3e-71bdbe4a2e70.png
```

This will produce `32x32.png`, `128x128.png`, `128x128@2x.png`, `icon.icns`,
`icon.ico` and Android/iOS variants in this folder automatically.
